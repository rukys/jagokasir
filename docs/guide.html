Instalasi Aplikasi

Saat ini JagoKasir dapat dijalankan pada perangkat bersistem operasi Android. Berikut cara menginstalnya:

  1. Unduh file instalasi `.apk` resmi terbaru dari halaman **Releases** di repository GitHub JagoKasir.
  2. Buka file APK tersebut di smartphone Anda. Jika muncul peringatan keamanan tentang instalasi dari sumber tidak dikenal, aktifkan izin untuk file manager atau browser Anda.
  3. Ketuk **Install** dan tunggu hingga proses pemasangan selesai. Aplikasi siap digunakan!

Setup Akun Owner Pertama Kali

Saat pertama kali membuka aplikasi, Anda akan diarahkan ke layar **Onboarding** untuk mendaftarkan akun Owner toko Anda:

  • Nama Lengkap: Masukkan nama lengkap Anda sebagai pemilik toko.
  • PIN Masuk: Buat PIN rahasia (4 hingga 6 digit angka) untuk melindungi data toko dari akses staff kasir biasa.
  • Setelah terdaftar, Anda dapat langsung login menggunakan profil tersebut.

Catatan: Hak akses Owner memiliki otoritas penuh untuk menambah Admin/Kasir baru di menu Pengaturan > Kelola Staff.

Menyambungkan Printer Thermal

Untuk mencetak struk belanja, sambungkan printer thermal Anda dengan mengikuti langkah-langkah berikut:

  1. Nyalakan Bluetooth pada printer thermal Anda serta pada smartphone/tablet Anda.
  2. Pasangkan (pair) perangkat printer thermal tersebut melalui pengaturan Bluetooth sistem smartphone Anda terlebih dahulu (masukkan PIN default printer jika diminta, biasanya `0000` atau `1234`).
  3. Buka aplikasi JagoKasir, masuk ke menu **Pengaturan > Pengaturan Printer**.
  4. Ketuk **Tambah Printer**, pilih jenis koneksi **Bluetooth**, dan cari nama printer thermal Anda yang telah dipasangkan.
  5. Pilih lebar kertas struk printer Anda (58mm atau 80mm), lalu simpan.
  6. Ketuk tombol **Cetak Test Struk** untuk memastikan koneksi printer berhasil berjalan dengan baik.

Ekspor & Impor Produk via CSV

Jika Anda memiliki ratusan produk, Anda dapat mengimpornya sekaligus secara massal menggunakan file template `.csv`:

  1. Buka menu **Produk** di aplikasi, ketuk ikon menu di pojok kanan atas, lalu pilih **Download Template CSV**.
  2. Buka file CSV tersebut di laptop menggunakan aplikasi spreadsheet seperti Google Sheets atau Microsoft Excel.
  3. Isi kolom template dengan benar:
    • name: Nama produk (Wajib).
    • sku: Kode SKU unik produk (Wajib).
    • selling_price: Harga jual (Wajib).
    • cost_price: Harga modal (Opsional).
    • barcode: Kode barcode produk (Opsional).
  4. Ekspor kembali tabel tersebut ke format **CSV (.csv)**.
  5. Kirim file tersebut ke smartphone Anda, buka menu Produk > ketuk **Import CSV**, dan pilih file tersebut untuk memasukkan semua produk secara otomatis.

Backup & Restore Database

JagoKasir menyimpan data Anda secara offline. Untuk mencegah data hilang jika perangkat Anda rusak atau hilang, lakukan pencadangan data secara berkala:

  • Backup Manual: Masuk ke menu **Pengaturan > Backup & Restore**, lalu ketuk **Buat Backup Baru**. File cadangan dalam format `.zip` (berisi enkripsi database) akan tersimpan di direktori penyimpanan telepon Anda.
  • Auto Backup: Aktifkan fitur pencadangan otomatis harian atau mingguan agar aplikasi mengamankan data secara terjadwal di latar belakang.
  • Restore: Untuk memulihkan data ke perangkat baru, salin file backup `.zip` Anda ke perangkat baru, buka menu Backup & Restore, ketuk **Restore Database**, lalu pilih file `.zip` tersebut. Aplikasi akan otomatis merefresh data.
