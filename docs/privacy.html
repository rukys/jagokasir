Komitmen 100% Offline & Lokal

Kami sangat menghormati hak privasi dan keamanan finansial bisnis Anda. Aplikasi JagoKasir Offline ini dibuat dengan prinsip privasi mutlak di mana pengembang Aplikasi tidak memantau, tidak mengoleksi, dan tidak memiliki sarana transmisi online apa pun.

Seluruh database berisi nama produk, harga modal, harga jual, rincian transaksi penjualan bulanan, jumlah stok gudang, dan kata sandi PIN staf dienkripsi/disimpan lokal pada partisi direktori penyimpanan internal yang aman pada perangkat smartphone Anda.

Tidak Ada Transfer Data ke Pihak Ketiga

Karena Aplikasi ini tidak menggunakan koneksi server eksternal, maka:

  • Tidak ada transmisi data penjualan Anda ke cloud pengembang atau pihak ketiga.
  • Tidak ada skrip analitik pelacakan perilaku (seperti Firebase Analytics, Facebook SDK, dll.) yang berjalan di latar belakang.
  • Tidak ada iklan berbasis minat atau penargetan lokasi geografi.

Transparansi Penggunaan Izin Perangkat

Untuk mendukung fitur fungsional operasional kasir fisik, Aplikasi memerlukan izin akses sistem Android/iOS berikut:

  • Bluetooth & Lokasi: Digunakan hanya untuk memindai perangkat sekitar dan menghubungkan Aplikasi dengan printer thermal Bluetooth untuk mencetak struk belanja kertas secara instan.
  • Akses Penyimpanan / File: Diperlukan untuk menyimpan file ekspor cadangan database (.db) ke direktori unduhan (Downloads) perangkat, serta membaca file cadangan tersebut pada saat Anda melakukan pemulihan data (Restore).
  • Kamera Perangkat: Diperlukan untuk memfungsikan lensa kamera HP Anda sebagai pemindai barcode / kode SKU produk secara otomatis dari kemasan barang di masa depan.

Penghapusan & Hak Data Anda

Semua data adalah milik mutlak Anda. Jika Anda ingin menghapus seluruh data transaksi dan produk secara permanen, Anda dapat:

  1. Melakukan uninstall / hapus pemasangan Aplikasi dari perangkat smartphone Anda (sistem operasi HP secara otomatis menghapus direktori internal penyimpanan database lokal Aplikasi).
  2. Melakukan hapus cache / hapus data Aplikasi melalui menu setelan sistem operasi HP Anda.

Perubahan Kebijakan Privasi

Pengembang dapat memperbarui Kebijakan Privasi ini sewaktu-waktu sesuai pengembangan fitur. Pembaruan kebijakan akan selalu mengedepankan keamanan data offline dan komitmen kami untuk menjaga data UMKM tersimpan secara privat di dalam HP pengguna masing-masing.
