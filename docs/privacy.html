Kebijakan Privasi JagoKasir

Kami sangat menghargai privasi Anda. Halaman ini menjelaskan bagaimana aplikasi **JagoKasir** mengelola dan menjaga keamanan informasi usaha Anda. JagoKasir dirancang sebagai aplikasi kasir (POS) **offline-first**, yang berarti kami tidak mengumpulkan, menyimpan, atau mengirimkan data transaksi Anda ke server eksternal mana pun.

Penyimpanan Data Lokal

Seluruh data usaha Anda yang dimasukkan ke dalam aplikasi JagoKasir, termasuk namun tidak terbatas pada:

  • Daftar nama produk, SKU, barcode, harga beli, dan harga jual.
  • Daftar kategori produk.
  • Riwayat dan laporan transaksi penjualan harian/bulanan.
  • Data akun staff (Owner, Admin, Kasir).
  • Konfigurasi printer struk belanja.

Tersimpan secara lokal di dalam database relasional **SQLite** pada memori penyimpanan internal perangkat (smartphone/tablet) Anda sendiri. Kami tidak memiliki akses untuk membaca, menyalin, atau menghapus data-data tersebut.

Izin Aplikasi (Permissions)

JagoKasir memerlukan beberapa izin akses pada perangkat Anda agar fitur-fitur aplikasi dapat berfungsi secara optimal:

  1. Bluetooth & Lokasi (Bluetooth Serial): Diperlukan untuk mendeteksi, menghubungkan, dan mengirimkan data cetak struk belanja ke printer thermal Bluetooth.
  2. Kamera: Diperlukan untuk memindai barcode/QR Code produk secara instan melalui modul kamera perangkat Anda.
  3. Penyimpanan Eksternal (Storage): Diperlukan saat Anda melakukan ekspor laporan ke format PDF/CSV atau membuat file cadangan (backup) database secara manual ke kartu memori.

Keamanan PIN Akun

Aplikasi ini mengamankan akses login staff Anda dengan mekanisme sandi **PIN (Personal Identification Number)**. Sandi PIN Anda tidak disimpan dalam bentuk teks biasa, melainkan disandikan menggunakan algoritma satu arah **BCrypt** yang sangat kuat langsung di database lokal Anda. Hal ini memastikan PIN tidak dapat dibaca kembali secara langsung oleh pihak mana pun.

Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu seiring dengan penambahan fitur-fitur baru pada aplikasi. Perubahan kebijakan akan diumumkan melalui pembaruan deskripsi pada halaman repository GitHub kami.

Jika ada pertanyaan lebih lanjut mengenai pengelolaan data lokal pada JagoKasir, silakan hubungi tim pengembang melalui halaman issue GitHub resmi kami.
